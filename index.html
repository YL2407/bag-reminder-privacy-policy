Privacy Policy For Bag Reminder App

This is a policy governing the interactions between Decycle Labs (us), our app "Bag Reminder", and the app's user (you). In using our application, you indicate agreement with and understanding of all of the terms and information laid out below.

Location Data Collection

The only data collected by this app is location data
Any data collected by this application is not stored (permanently) in any of our databases. The only use which this data is subjected to, necessary as part of its functioning, is that it is passed into a web-based API (Nominatim), from which, on the client-side, nearby supermarkets and malls are discovered. If you want to find out how your data is used on that API, refer to the OpenStreetMap Foundation Privacy Policy here.

The normal, day-to-day function of this application requires access to your location in the background, as the app must function even when it is not in the foreground so as to interfere minimally with your other activities. Once again, the only processing this data receives is passing it through the aforementioned API.
